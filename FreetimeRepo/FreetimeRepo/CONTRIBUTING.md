# Contributing to Freetime

We love community input! If you want to help Ben build the best social calendar app, here is how you can contribute.

## How to Contribute
1. **Fork the Repo:** Create your own fork and work on a new branch.
2. **Code Style:** We follow strict Apple Guidelines and Swift Format.
   - Use `View` and `ViewModel` (MVVM) separation.
   - Keep UI components small and reusable (SwiftUI).
3. **Pull Requests:** Submit your changes back to the `main` branch via a Pull Request (PR).

## Important: Intellectual Property (IP)
By submitting a Pull Request to this repository, you agree to transfer the copyright of your code contributions to **Ben Chytrek (Freetime)**. 
This transfer ensures that the project remains legally safe and can be published on the App Store without complex licensing issues. You will be credited as a contributor!
